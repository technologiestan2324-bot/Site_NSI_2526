Question 10

On se place dans le dossier important qui contient le fichier prog1.py. Le dossier TP se trouve dans le même dossier parent.

En utilisant un chemin absolu à partir du dossier parent de important, on peut écrire : 

cp /important/prog1.py /TP/prog1.py

Cette commande copie le fichier prog1.py depuis le dossier important vers le dossier TP.

Question 14

Le protocole de routage utilisé est RIP.

Justification : le protocole RIP choisit la route comportant le plus petit nombre de routeurs (nombre de sauts), sans tenir compte du débit des liaisons.

Question 15

Le coût OSPF est calculé avec la formule : 

C = 10⁸ / BP
  • 100 Mb/s : 10⁸ / (100 × 10⁶) = 1
  • 1 Gb/s : 10⁸ / 10⁹ = 0,1
  • 10 Gb/s : 10⁸ / 10¹⁰ = 0,01

Question 16

Le protocole OSPF choisit la route dont la somme des coûts est minimale.

Chemin choisi : 

R1 → R5 → R4 → R3 → R8 → R9

Calcul du coût total : 

0,01 + 0,1 + 0,1 + 0,01 + 0,1 = 0,32

Ce chemin est retenu car son coût total est inférieur à celui des autres chemins possibles.
